手把手教你快速搭建Hexo+Github静态博客

Posted on 2015-03-06

安装GIT

下载 msysgit 安装。

安装Node.js

windows环境下，下载 nodeJS 安装。

安装hexo

利用 npm 命令安装hexo。（在任意位置点击鼠标右键，选择Git bash）

npm install -g hexo

创建hexo文件夹

在你喜欢的位置新建hexo文件夹（如F:/hexoblog），在文件夹内右键Git Bash，初始化hexo

hexo init

安装依赖包

npm install

本地预览

至此hexo的全部本地环境已搭建完毕，在项目文件根目录执行以下命令，在localhost:4000预览效果

hexo generate
hexo server

将hexo部属到github

注册一个github帐号

用户名不要太复杂，github默认域名username.github.io)

登录后创建一个新的repository（仓库）

仓库名一定要以 username.github.io 为格式，如用户名为guowc，则仓库名guowc.github.io

本地部署

在项目根目录下找到 _config.yml 文件，在文件末尾找到deploy字段，配置你的github信息(guowc替换为你的用户名)

deploy:
  type: github
  repository: https://github.com/guowc/guowc.github.io.git
  branch: master

修改完成后，试试发布一篇新文章！
hexo new “hello hexo!” //创建一篇新文章
hexo generate //编译(每次修改本地文件都需编译后发布)
hexo deploy //发布

访问username.github.io看看你的成就！

hexo常用命令

hexo已支持短命令，如 hexo d 代替 hexo deploy

hexo g //编译
hexo d //发布
hexo s //启动本地预览服务
hexo n //创建一篇新文章
